# Simple NFC Tools

Browser-based NFC utilities powered by [WebNFC](https://developer.mozilla.org/en-US/docs/Web/API/Web_NFC_API). Hosted on GitHub Pages — no install needed.

## Tools

### UID Collector

Scan NFC tokens one after another, automatically de-duplicate, and copy all unique UIDs to your clipboard.

## Requirements

- **Chrome on Android** (WebNFC is not supported on iOS or desktop browsers)
- **HTTPS** (provided by GitHub Pages)

## Usage

Open the hosted site, tap "Start Scanning", and hold NFC tokens to your phone one by one.
